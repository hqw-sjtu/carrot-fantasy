## 简介

一款基于Python和Pygame开发的塔防游戏，灵感来源于经典游戏"保卫萝卜"。

## 操作指南

- **1-5**: 选择防御塔（箭塔/炮塔/魔法塔/减速塔/冰霜塔）
- **鼠标点击**: 放置防御塔
- **R**: 切换攻击范围显示
- **H**: 切换血量显示
- **T**: 切换统计面板
- **Q/W/E**: 技能按键（减速/冰冻/群攻）
- **U**: 升级选中塔
- **D**: 出售选中塔
- **P**: 切换攻击优先级
- **S**: 打开商店
- **F5**: 截图
- **F9**: 快速保存
- **F10**: 快速读取
- **ESC**: 暂停/继续
- **Tab**: 切换速度

## 运行方式

```bash
pip install -r requirements.txt
python launcher.py
```
