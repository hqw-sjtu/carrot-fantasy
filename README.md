# 🏰 保卫萝卜 (Carrot Fantasy)

> 基于 Python + Pygame 的塔防游戏 | 工艺品级别品质

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)
[![Pygame](https://img.shields.io/badge/Pygame-2.0+-green.svg)](https://www.pygame.org/)

## ✨ 特色功能

### 🎯 核心玩法
- **6种防御塔**: 箭塔、炮塔、魔法塔、减速塔等
- **15+波次挑战**: 多种难度级别
- **升级系统**: 最高3级，支持技能树
- **合成系统**: 同类塔合成高阶塔

### 🎨 特效系统
- **粒子系统**: 攻击、爆炸、升级光晕
- **伤害数字**: 暴击、连击显示
- **暴击系统**: 暴击伤害放大、星星爆发、光晕描边效果 💥
- **连击系统**: 连续击杀显示连击数+金币奖励
- **升级爆发**: 升级时圆形光圈+粒子飞散特效
- **塔基特效**: 攻击时发光环、粒子环
- **暴击闪光**: 十字闪光反馈
- **连锁闪电**: 魔法塔链式攻击
- **屏幕震动**: 强力攻击时震动
- **协同加成**: 塔与塔之间产生组合特效
- **共鸣系统**: 同类型塔靠近时触发视觉特效+伤害加成(最高25%)
- **热力图系统**: 敌人密度分布、防御塔范围指示、路径可视化 📡
- **Boss战斗阶段**: 警告闪烁、暗角效果、屏幕震动、战斗/胜利粒子 ⚔️

### 📊 任务与成就
- 每日任务系统
- 成就徽章系统
- 统计追踪

### 🐾 宠物系统
- 可爱宠物伴随冒险(小猫/小狗/小兔)
- 专属增益效果
- 跟随动画+弹跳效果

## 🕹️ 安装与运行

```bash
# 克隆项目
git clone https://github.com/hqw-sjtu/carrot-fantasy.git
cd carrot-fantasy

# 安装依赖
pip install pygame pytest

# 运行游戏
python src/main.py

# 运行测试
pytest tests/
```

## 🎮 操作说明

| 操作 | 功能 |
|------|------|
| 鼠标点击 | 放置/升级防御塔 |
| 数字键1-4 | 快速选择塔类型 |
| 数字键1/2/3 | 切换速度(慢放/正常/快进) |
| Tab | 循环切换速度/波次预览 |
| Space | 暂停/继续 |
| F9 | 快速保存 |
| F10 | 快速读取 |
| Esc | 退出 |

## 📁 项目结构

```
carrot-fantasy/
├── src/           # 源代码
│   ├── main.py           # 主游戏逻辑
│   ├── towers.py         # 防御塔系统
│   ├── monsters.py       # 怪物系统
│   ├── projectiles.py    # 子弹系统
│   ├── particle_system.py # 粒子系统
│   ├── base_effects.py   # 塔基特效系统 ⭐
│   └── ...
├── tests/         # 测试代码
├── config.json    # 游戏配置
└── docs/          # 项目文档
```
