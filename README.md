# 保卫萝卜 (Carrot Fantasy)

> A Tower Defense Game

## 项目简介

保卫萝卜是一款经典塔防游戏，玩家需要在地图上建造各种防御塔来阻止怪物到达萝卜。

## 技术栈

- Python 3.x
- Pygame

## 安装

```bash
pip install -r requirements.txt
```

## 运行

```bash
python src/main.py
```

## 项目结构

```
carrot-fantasy/

├── assets/           # 资源文件
```
